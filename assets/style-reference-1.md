整体风格参考：中文财经/科技媒体 LatePost 式极简数据可视化海报，3:4 竖版，适合手机阅读。背景使用米白色旧纸质感（#F8F7F1 / #F8F7F4），带轻微印刷颗粒和纸纹，但保持干净留白。视觉语言克制、理性、 editorial，不使用渐变、不使用阴影、不使用 3D、不使用拟物、不使用装饰性插画。

版式上采用大留白、强标题、少色彩、高对比。标题通常位于下半部或顶部左侧，使用粗黑现代中文无衬线字体，字号大、字重重，层级清晰；副标题和来源信息使用浅灰色小字。图表主体使用抽象线条、流线、横向条形图、时间线、矩阵、lollipop chart、数字卡片等平面形式，避免复杂图标和花哨元素。

配色以米白背景、黑色/深炭色主信息、灰色辅助信息、正红色高亮为核心：
- 背景：#F8F7F1 或 #F8F7F4
- 主黑：#1A1A1A
- 强调红：#C92816 或 #D62828
- 中灰：#8E8E8A
- 浅灰：#B6B3AA
可以少量加入 muted teal 作为次级区分色，但整体不要变成多彩信息图。

线条设计应像精致印刷图表：贝塞尔曲线、细线、端点圆点、简洁节点、严谨比例。红色只用于最重要的一条线、一个数据或一个结论；黑色用于主结构；灰色用于背景流动和次要信息。图形应该像“严肃研究报告的封面/内页图表”，而不是营销海报。

强约束：无 logo、无水印、无旗帜、无地图、无人物肖像、无政治/宗教/军事符号。所有文字必须清晰可读，数字必须精确，图表比例必须与数据一致。画面应有高级纸媒感、冷静分析感、少即是多的张力。
